Predict Exercise Action

Executive Summary

This report attempts to predict the manner in which people perform weight lifting exercise. Data for this analysis is obtained from http://groupware.les.inf.puc-rio.br/har. Six young health participants were asked to perform one set of 10 repetitions of the Unilateral Dumbbell Biceps Curl in five different fashions: exactly according to the specification (Class A), throwing the elbows to the front (Class B), lifting the dumbbell only halfway (Class C), lowering the dumbbell only halfway (Class D) and throwing the hips to the front (Class E).

Loading & Cleaning Data

pml_training <- read.csv("pml-training.csv")
pml_testing <- read.csv("pml-testing.csv")

Creating training & cross-validation dataset:

library(caret)

## Loading required package: lattice
## Loading required package: ggplot2

set.seed(123)
inTrain <- createDataPartition(y=pml_training$classe,
                              p=0.75, list=FALSE)
training <- pml_training[inTrain,]
testing <- pml_training[-inTrain,]

Remove zero variance variables

nzv <- nearZeroVar(training)
training <- training[-nzv]
testing <- testing[-nzv]
testing_final <- pml_testing[-nzv]

Remove variables like username and timestamp which are not dependent variable Impute values

pre_model <- preProcess(training[,-c(1,2,3,4,5,6,101)],method = c("knnImpute"))
prediction_train <- cbind(training$classe, predict(pre_model,training[,-c(1,2,3,4,5,6,101)]))
prediction_test <- cbind(testing$classe, predict(pre_model, testing[,-c(1,2,3,4,5,6,101)]))
prediction_final_test <- predict(pre_model, testing_final[,-c(1,2,3,4,5,6,101)])
names(prediction_train)[1] <- "classe"
names(prediction_test)[1] <- "classe"

Random Forest Model

Prediction model using Random Forest

library(randomForest)

## randomForest 4.6-10
## Type rfNews() to see new features/changes/bug fixes.

forest_model <- randomForest(classe ~ ., prediction_train, mtry=32)
